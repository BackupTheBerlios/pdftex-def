pdftex.def

This project develops the pdftex driver for the graphics package for use with LaTeX and plain TeX (with the help of miniltx).

The berlios page can be found here.

The current stable release is v0.04e: tgz and zip

You can browse the CVS repository.

We have three mailing lists:

  • pdftex-def-announce for announcements regarding pdftex.def
  • pdftex-def-dev for discussions on the development of pdftex.def
  • pdftex-def-cvs automatically gets the CVS-commits of pdftex.def

Questions regarding the usage of pdftex.def or pdftex should be directed to the pdfTeX list.

Usage statistics for this page can be found here.
